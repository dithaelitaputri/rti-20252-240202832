# Rencana Penelitian: Analisis Usability dan Continuance Intention TikTok Shop

## 1. Ringkasan

| Item | Keterangan |
|---|---|
| Judul | Analisis Usability dan Continuance Intention Fitur Live Shopping dan Product Reviews pada Platform TikTok Shop |
| Target Publikasi | Jurnal Ilmiah Terakreditasi SINTA |
| Stack | Google Form, Microsoft Excel / SPSS / Python, CSV Data Store |
| Masalah | Adanya keluhan pengguna mengenai kepadatan elemen visual antarmuka TikTok Shop yang berpotensi membingungkan proses transaksi pengguna. |
| Solusi | Mengukur kelayakan antarmuka menggunakan *System Usability Scale* (SUS) dan menguji dampaknya secara kuantitatif terhadap niat loyalitas keberlanjutan belanja (*Continuance Intention*). |
