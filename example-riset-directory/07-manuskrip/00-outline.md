# Outline dan Klaim Kunci Manuskrip Jurnal (Tahap 7)

Dokumen ini memetakan struktur penulisan artikel ilmiah berbasis data evaluasi fitur TikTok Shop.

## 1. Spesifikasi Naskah Jurnal
*   **Target Publikasi**: Jurnal Sistem Informasi / Jurnal Ilmiah SINTA-terakreditasi
*   **Objek Penelitian**: Sistem Antarmuka Fitur *Live Shopping* dan *Product Reviews* TikTok Shop
*   **Metode Evaluasi**: *System Usability Scale* (SUS) & *Continuance Intention* (CI) Analysis
*   **Ukuran Sampel ($N$)**: 115 Responden Valid

## 2. Peta Klaim Kunci Penelitian
Setiap bab di dalam draf naskah wajib konsisten mempertahankan tiga temuan utama ini:
1.  **Skor Usability Marginal**: Baik fitur *Live Shopping* (62.72) maupun *Product Reviews* (62.52) berada di bawah ambang batas standar kelayakan global SUS (skor 68). Keduanya mendapat predikat **Grade D (Marginal Low)**.
2.  **Retensi Pengguna Tinggi**: Meskipun antarmukanya dinilai cukup rumit dan membingungkan, tingkat niat keberlanjutan (*Continuance Intention*) pengguna untuk berbelanja di masa depan sangat tinggi, yaitu mencapai **86.96%** (100 responden).
3.  **Anomali Perilaku Konsumen**: Terjadi korelasi paradoks di mana rendahnya kualitas *usability* tidak menurunkan niat transaksi. Hal ini dipicu oleh faktor retensi nilai eksternal (diskon, promo ongkir, daya tarik interaksi *live host*) yang mengompensasi kekurangan sistem antarmuka.
